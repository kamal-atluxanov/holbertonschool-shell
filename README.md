# Shell Scripting Basics

This repository contains resources and exercises for learning **Shell scripting** and command-line operations on Unix/Linux systems.


##  What is Shell?

A **Shell** is a command-line interface that allows you to interact with the operating system. 
Shell scripting enables **automation of tasks**, running commands in sequence, and managing system operations efficiently.


##  Key Concepts

### 1. **Basic Commands**
- Navigation: `cd`, `ls`, `pwd` 
- File operations: `cp`, `mv`, `rm`, `mkdir` 

### 2. **Variables**
- Storing data for reuse in scripts 
- Example: `name="Kamal"` and using `$name`

### 3. **Control Flow**
- Conditional statements: `if`, `else`, `elif` 
- Loops: `for`, `while`, `until`

### 4. **Functions**
- Group commands into reusable blocks 
- Call functions with arguments

### 5. **Input & Output**
- Read input from users: `read` 
- Print output: `echo` 
- Redirect output: `>` and `>>`

### 6. **Automation**
- Writing scripts to automate backups, log checks, and system maintenance
- Make scripts executable: `chmod +x script.sh`


##  Learning Goals

- Understand basic shell commands and scripting
- Automate repetitive system tasks 
- Write simple to intermediate shell scripts 
- Improve productivity in Unix/Linux environments


**Tip:** Start with interactive shell commands, then gradually write scripts for automation and task management.
